Mac+Hexo+GitHub博客搭建

发表于 2018-11-13

前言

积土成山，风雨兴焉；积水成渊，蛟龙生焉

很早之前就有搭建自己博客的想法，加上最近找实习的时候意识到了技术博客的重要，于是有了这第一篇博客。

目前github博客主流是Jekyll和Hexo两种搭建方式。
Hexo更加简洁美观，能通过命令实现本地预览，并直接发布到web容器实现同步。
